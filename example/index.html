The header above is enabled with the directive content-editable and you can correct it by clicking.
This is a normal non-editable paragraph instead.

This paragraph has the focus-select attribute, when you click it, it auto select all text on focus.
So, you can easily edit it by clicking it and pressing Backspace to erase all text selected.
If you make a mess and you want to undo the changes simply press Esc and rollback to previous value.

You can also add the attribute render-html and allow the directive to compile any html text you entered.
